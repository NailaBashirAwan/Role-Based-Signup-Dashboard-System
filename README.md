You can run the assignment project by following the given steps:
1- unzip the Assignment Task folder.
2- open Assignment Task folder in VS code.
3- activate the virtual environment env. use command ----env/Scripts/activate----
4- move to the project directory named role_system use command ----cd role_system----
5- Run the server use command ----python manage.py runserver-----
6- Open the server link in browser. http://127.0.0.1:8000/
7- Quit the server with CTRL-BREAK.
